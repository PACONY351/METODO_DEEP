Día 1 - Configuración y Fundamentos Iniciales

12 de Mayo, 2025 - C, C++ y C#

Introducción

Objetivo: Preparar el entorno de desarrollo y aprender los conceptos más básicos de C, C++ y C#.

Consejo: Dedica tiempo a configurar bien tu entorno, esto te ahorrará problemas más adelante.

1. Configuración del entorno (1-2 horas)

Tareas:

  • Para C/C++:
    • Windows: Instalar MinGW o Visual Studio (con workload "Desarrollo para C++")
    • Linux/macOS: Usar gcc y g++ (ya vienen instalados en la mayoría de sistemas)
  • Para C#:
    • Instalar Visual Studio (Community Edition) con workload ".NET Desktop Development"
    • Alternativa: .NET SDK + VS Code
# Para C:
gcc --version

# Para C++:
g++ --version

# Para C#:
dotnet --version
  • VS Code con extensiones:
    • C/C++ (Microsoft)
    • C# (OmniSharp)

2. Primeros pasos en C (2-3 horas)

Temas y subtemas:

  • Estructura de un programa en C (main(), #include <stdio.h>)
  • Comentarios (// y /* */)

  • int, float, char, double
  • Declaración e inicialización

  • printf() para imprimir
  • scanf() para leer datos

Ejercicios prácticos:

🛠 1. "Hola Mundo" en C:

#include <stdio.h>
int main() {
    printf("Hola, mundo!\n");
    return 0;
}

🛠 2. Programa que suma dos números:

#include <stdio.h>
int main() {
    int a, b;
    printf("Ingresa dos números: ");
    scanf("%d %d", &a, &b);
    printf("La suma es: %d\n", a + b);
    return 0;
}

Objetivos:

  • Entender la estructura básica de un programa en C
  • Aprender a declarar variables y usar printf/scanf

5. Proyecto Final del Día (1-2 horas)

Mini-Proyecto: "Calculadora Básica" en los 3 lenguajes

Objetivo: Aplicar lo aprendido en un proyecto sencillo.

Consigna:

Crear un programa que pida dos números y realice las 4 operaciones básicas (+, -, *, /).

En C

Usar printf/scanf y funciones separadas para cada operación.

En C++

Usar cin/cout y sobrecarga de funciones (opcional).

En C#

Usar Console.ReadLine() y switch-case para elegir la operación.
